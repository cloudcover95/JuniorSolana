# JuniorSolana Architecture

## Layers
1. Swift UI + Quick Actions (deep linking)
2. Solana program client (wallet connect, transactions)
3. BitNet-Intel agentic layer (on-device intelligence)
4. Python bridge (JuniorPython) for complex on-chain + AGI logic

## Key Features
- Apple-style Quick Actions for wallet/app linking
- BitNet 1.58 × 3.0 for efficient on-device reasoning
- Sovereign self-host option via JuniorHome deployment stack
